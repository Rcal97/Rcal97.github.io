01

SQL Complexity Classifier

Thesis project

Hybrid machine learning classifier that predicts SQL query complexity by combining 6 handcrafted syntactic features with 128-dimensional TinyBERT semantic embeddings. Trained and validated on the Spider NL2SQL benchmark, achieving 94.48% accuracy (F1 = 0.9448) with an SVM Linear classifier. Applied diagnostically to 253 failures of the state-of-the-art C3 model, confirming that structural complexity reliably predicts NL2SQL system breakdowns.
